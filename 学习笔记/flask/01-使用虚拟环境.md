# 使用虚拟环境

## 步骤

1.创建虚拟环境

```shell
pip install virtualenv
virtualenv 环境名
cd 环境名
virtualenv -p python路径 环境名,来创建虚拟环境
```

2.进入环境下的Scripts文件夹下

```shell
activate
接下来就可以 在该虚拟环境下pip安装或者做各种事情
```

3.退出虚拟环境

```shell
deactivate
```

## 管理虚拟环境

```shell
pip3 list 列出环境的列表
pip install virtualenvwrapper
mkvirtualenv --python=python路径名 环境名
lsvirtualenv 列出环境名
workon 环境名
pip3 install flask==1.0(卸载原来的版本，安装指明的版本)
```
